# 📝 文档对比与编辑助手

完全**本地运行**的文档对比与编辑工具，基于 Python + Streamlit 构建，不调用任何外部 API，确保文档隐私安全。

## 功能特性

| 功能 | 说明 |
|------|------|
| 多格式上传 | 支持 `.docx`、`.md`、`.txt`，一次上传 2~3 个文件 |
| 智能预处理 | Toggle 开关：忽略大小写 / 忽略空白行 / 忽略注释 |
| 并排对比视图 | GitHub 风格，左右面板逐行对齐，支持单词级差异高亮 |
| 行内对比视图 | Unified diff 风格，`+` / `−` 标注增删行 |
| 差异统计 | 相似度、行数、新增行数、删除行数一览 |
| 在线编辑器 | 选择任一文件进行修改，支持重置为原始内容 |
| 导出 | 导出为 `.md` 或 `.txt` |

## 快速开始

### 1. 安装依赖

```bash
pip install -r requirements.txt
```

> 如果不需要处理 `.docx` 文件，可仅安装 `streamlit`：
> ```bash
> pip install streamlit
> ```

### 2. 启动应用

```bash
streamlit run app.py
```

浏览器会自动打开 `http://localhost:8501`。

## 使用说明

1. **上传文件**：在页面顶部上传 2 或 3 个文件（支持混合格式）。
2. **预处理**（可选）：在左侧边栏启用所需的规范化选项。
3. **选择对比对**：上传 3 个文件时，通过下拉框选择要对比的文件对。
4. **查看差异**：
   - 并排视图：左右面板逐行对齐，黄色高亮表示修改行（单词级），红色表示删除，绿色表示新增。
   - 行内视图：`−` 红色为删除行，`+` 绿色为新增行。
5. **编辑**：在编辑器中选择基础文件进行修改。
6. **导出**：点击导出按钮，下载 `.md` 或 `.txt` 文件。

## 注释过滤规则

| 注释类型 | 示例 | 是否过滤 |
|----------|------|----------|
| HTML / Markdown 块注释 | `<!-- 注释 -->` | ✅ |
| LaTeX 行注释 | `% 注释内容` | ✅ |

## 隐私说明

- 所有文件读取和差异计算均在本地内存完成。
- 使用 Python 内置 `difflib` 库，零外部依赖（除 Streamlit 和可选的 python-docx）。
- 不收集、不上传任何文档内容。
